# making-premiere-directory-script

Adobe Premiere Pro のディレクトリ作成を一括で行うスクリプト

ディレクトリ構成
sh01-first - 作成する動画のフォルダ
01_footage - 動画データ
02_asset   - 動画以外の素材
 img       - 画像素材
 se        - 効果音素材
 bgm       - BGM素材
03_proj    - Premiereデータ
04_out     - 動画書き出し用

```
.  
⎿＿ sh01-first  
　　　├─ 01_footage  
　　　├─ 02_asset  
　　　│　├─  img  
　　　│　├─  se  
　　　│　└─  bgm  
　　　├─ 03_proj  
　　　└─ 04_out
```
